# nattachments

Universal file attachments component — upload, delete, reorder, rename, lightbox, PDF preview.

## Key Classes
- `AttachmentsControl` — Nette UI component for file management
- `AttachmentsControlFactory` — DI factory interface

## Conventions
- PHP 8.2+ with `declare(strict_types=1)`
- PSR-4 autoloading in `src/`
- Nette DI Extension for service registration
- PHPStan level 9
- Part of `jansuchanek/*` on Packagist
- GitHub: https://github.com/JanSuchanek/nattachments
